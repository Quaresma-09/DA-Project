Detailed Description

Builds and solves the reviewer assignment problem using max-flow.

Constructs a flow network where reviewers are matched to submissions based on domain expertise, and computes a valid assignment using the Edmonds-Karp algorithm. Supports different assignment modes:

  • Mode 0/1: Primary domains only.
  • Mode 2: Primary + Secondary submission domains, Primary reviewer expertise.
  • Mode 3: All primary and secondary domains of both.

\par Complexity: O(V * E^2) for the max-flow computation.
