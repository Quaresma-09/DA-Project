Detailed Description

Handles the extraction of data from a formatted CSV file.

  • The class processes four main sections: Submissions, Reviewers, Parameters, and Control. It ensures data integrity by checking for duplicate IDs and ignoring comments or empty lines.

◆ parse()

bool CsvParser::parse ( )

Orchestrates the parsing process of the CSV file.

Main parsing loop using a State Machine approach.

  • Reads the file line by line, identifies sections, and populates the corresponding data structures (Submissions, Reviewers, Config).
  • Returns
    true if the file was parsed successfully without critical errors, false if the file could not be opened or has severe formatting issues.
    The method identifies headers (e.g., Submissions) to switch states and determines how to process subsequent lines based on the current state. Includes error handling for duplicated IDs, malformed integers, and validation of mandatory parameters.
Returns
true if the file was parsed successfully, false otherwise. \par Complexity: O(N) where N is the number of lines in the file.

Internal states to track which section of the CSV is being read.
