Detailed Description

Implements the Edmonds-Karp algorithm for maximum flow.

Uses BFS to find augmenting paths in the residual graph and updates the flow until no more augmenting paths exist. This is a BFS-based variant of the Ford-Fulkerson method.

Finds an augmenting path from source to sink using BFS.

Traverses the residual graph looking for a path with available capacity. Populates parent and parentEdge vectors for path reconstruction.

Computes the maximum flow from source to sink using Edmonds-Karp.

Repeatedly finds augmenting paths via BFS and pushes the maximum possible flow along each path until no more paths exist.
